                                  Лабораторная работа №12
Задание 1.
При нажатии на вкладку Подписки открывается новое окно с некоторым текстом.
Данная задача реализована с помощью файла window.js.
Задание 2.
Во вкладке Подписки, в подпункте Индивидуальный находится страница (individual.html)
для заполнения формы с проверкой на корректность ввода данных пользователем.
После заполнения формы и нажатия на кнопку Отправить, перед экраном
появляется диалоговое окно с надписью: "В течении дня с вами свяжется оператор".
Задание 3.
При нажатии на кнопку Заявка на экране появится данные введенные пользователем 
в форме и статус заявки.

                               Лабораторная работа №13
Задание 1.
Во вкладке О нас, в подпункте Наши филиалы находится карта Казахстана, на котором 
можно посмотреть филиалы по всей стране, на данный момент филиалы находятся в 5 городах
Казахстана: Петропавловск, Астана, Алматы, Шымкент, Атырау. При наведении на эти города
рядом появляется краткая информация о этих филиалах: город, адрес, график работы и номер
телефона.
Задание 2.
Во вкладке О нас, в подпункте О нас (about us.html) находятся два текстовых блока.
В первом текстовом блоке, при наведении курсора мыши на слово "QTEST" снизу появляется изображение.
Во втором текстовом блоке, при наведении курсора мыши на слово "ЕНТ" снизу, также появляется изображение.

                               Лабораторная работа №14
Во вкалдке О нас, в подпункте Наши филиалы под картой Казахстана находится круговая
диаграмма, которая показывает число свободных мест в разных филиалах.
В диаграмме реализована анимация и всплывающая подсказка.
